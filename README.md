# Login App 🔐

Website login dengan audio feedback menggunakan Web Audio API.

## 🌐 Link Website

```
https://barraqieazka7-creator.github.io/login-app
```

## ✨ Fitur

✅ Interface login modern
✅ Audio feedback dengan Web Audio API
✅ Responsive design (mobile, tablet, desktop)
✅ HTTPS otomatis via GitHub Pages
✅ Tanpa dependencies eksternal
✅ Audio dijamin kedengeran! 🔊

## 📱 Cara Pakai

1. Klik tombol **"Masuk"** → Audio naik (C → E → G) 🎵
2. Klik tombol **"Keluar"** → Audio turun (G → E → C) 🎵

## 🎵 Audio

Audio dibuat real-time menggunakan Web Audio API:
- Tidak perlu file MP3/WAV
- Kompatibel semua browser modern
- Suara dijamin keluar! ✅

## 🛠️ Tech Stack

- HTML5
- CSS3 (Gradient, Animation)
- JavaScript (Web Audio API)
- GitHub Pages (Free Hosting + HTTPS)

## 🚀 Deployment

Otomatis deploy ke:
```
https://barraqieazka7-creator.github.io/login-app
```

Gunakan Settings → Pages untuk enable GitHub Pages jika belum aktif.
